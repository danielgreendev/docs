# ERC20BatchMintable

Enable minting multiple NFTs at once in a single transaction by Implementing `ERC20`, `IMintableERC20`, and `Multicall`.

It is an extension which is detectable in the dashboard if the smart contract implements the 
[ERC20](/solidity/extensions/erc20), [Multicall](/solidity/extensions/multicall) and [ERC20Mintable](/solidity/extensions/interfaces/erc20mintable) extensions.

## Implementing It Yourself

This is an example smart contract which will have ERC20BatchMintable detected on it in the dashboard.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/eip/ERC20A.sol";
import "@thirdweb-dev/contracts/extension/interface/IMintableERC20.sol";
import "@thirdweb-dev/contracts/extension/Multicall.sol";

contract Contract is ERC20A, IMintableERC20, Multicall {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC20A(
            _name,
            _symbol
        )
    {}

    function mintTo(address to, string calldata uri) external override returns (uint256) {
        // Your custom implementation here
    }

}
```
