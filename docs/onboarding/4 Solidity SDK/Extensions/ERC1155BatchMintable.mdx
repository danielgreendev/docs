# ERC1155BatchMintable

Enable minting multiple NFTs at once in a single transaction by Implementing `ERC1155`, `IMintableERC1155`, and `Multicall`.

It is an extension which is detectable in the dashboard if the smart contract implements the 
[ERC1155](/solidity/extensions/erc1155), [Multicall](/solidity/extensions/multicall) and [ERC1155Mintable](/solidity/extensions/erc1155mintable) extensions.

## Implementing It Yourself

This is an example smart contract which will have ERC1155BatchMintable detected on it in the dashboard.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@thirdweb-dev/contracts/eip/ERC1155.sol";
import "@thirdweb-dev/contracts/extension/interface/IMintableERC1155.sol";
import "@thirdweb-dev/contracts/extension/Multicall.sol";

contract Contract is ERC1155, IMintableERC1155, Multicall {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC1155(
            _name,
            _symbol
        )
    {}

    function mintTo(
        address to,
        uint256 tokenId,
        string calldata uri,
        uint256 amount
    ) external override {
        // Your custom implementation here
    }
}
```
